# Recorder

--------------------------------------------------
## Overview · 项目概述

知识技术的学习和应用是一个长期的持久化的过程，同一个问题会随着个人知识技术的积累而呈现多样的解决方案。**Recorder** 是我的个人项目，目的是通过记录个人知识技术体系的实际建立过程来探索其方法论。
